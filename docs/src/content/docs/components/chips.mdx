<div class="chips-mixin-wrapper">
  <h2 class="chips-mixin-heading">Mixin</h2>

    Chips can be created with a sass mixin. The mixin takes a color Map (defaults
    to `$skys`) as an input parameter. The chip background is given the 10
    lightness of the color and text is given the 120 lightness.

</div>

```scss
@use "openaq-design-system/variables" as variables;
@use "openaq-design-system/mixins" as mixins;

@include mixins.chip(variables.$oceans);
```
