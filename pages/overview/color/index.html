Color

Primary color palette

The full color palette provides a wide range of colors. Three colors should be used at the primary colors which are aliases for colors within the color palette:

Brand
Base data
Base interactive
Status OK
Status Alert
Status Warning
  • "Brand"/sky-100 - Brand is the primary color, matching the OpenAQ logo color
  • "Base data"/lavender-100 - Base data is the primary color for data, charts, etc
  • "Base interactive"/ocean-100 - Base interactive can be used as a secondary color for data, charts etc and as a interactive color
  • "Status OK"/mantis-100 x- Status OK is the primary color for "success" or general positive theming
  • "Status Alert"/corn-100 - Status Alert is the primary color for alerting or somewhat negative theming
  • "Status Warning"/fire-100 - Status Warning is the primary color for "failure" or general negative theming

Full color palette

The color system is based around 7 hues (sky, ocean, lavender, mantis, corn, fire smoke) and white with up to 9 levels of lightness per hue. Not all hues have all 9 levels defined. When overlaying text atop a color, white should be used on all color above the 30 lightness and smoke-180 or black should be used for lightness levels 30 and below (30,20,10).

SCSS

Variables

Colors are avaiable as SCSS variables in the form of hue name + lightness

$lavender180: #241050;
$lavender120: #584dae;
$lavender100: #6a5cd8;
$lavender80: #7867eb;
$lavender60: #8f81ee;
$lavender40: #a497fd;
$lavender30: #bcb2fe;
$lavender20: #cec7ff;
$lavender10: #dedafb;

Maps

Maps are available for all 7 hues, with keys referring to each lightness level:

$skys: (
  '120': $sky120,
  '100': $sky100,
  '60': $sky60,
  '30': $sky30,
  '10': $sky10,
);

$lavenders: (
  '180': $lavender180,
  '120': $lavender120,
  '100': $lavender100,
  '80': $lavender80,
  '60': $lavender60,
  '40': $lavender40,
  '30': $lavender30,
  '20': $lavender20,
  '10': $lavender10,
);

$oceans: (
  '120': $ocean120,
  '100': $ocean100,
